# POKEDESX APP
## React web app
This is a simple app which consumes the marvel API and return its data. it allow you to also do an instant search by characters name and navigate through its pagination system


![marvel](marvel.png)

### Technologies
* React.js
* React Hooks
* Styled-Components
* TypeScript
